# Project Brief

Proyecto: Asistencia y Despacho de Juguetes Dia del Nino.

Objetivo: migrar el flujo legado ASP.NET MVC a una solucion moderna con NestJS, React, Flutter y SQL Server, integrada con SSO Portal.

Flujos principales:

- importar censo desde Excel
- importar catalogo de juguetes desde Excel
- registrar asistencia por carnet
- despachar juguetes por hijo
- controlar stock con transacciones
- reversar entregas con auditoria
- exponer dashboard y reportes

Regla base: usar SQL Server con `mssql` directo, sin ORM.
